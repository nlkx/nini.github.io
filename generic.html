    Week 2

    What is Git?

    Git is an Open Sourced Distributed Version Control System. Is a software for developers to use to store codes, track revision history, merge code changes and revert to earlier codes if needed.

    What is Github?

    GitHub is a website and cloud-based service that helps developers store and manage their code, as well as track and control changes to their code.

    Python Koans

    What is Python Koans?

    Python Koans is an interactive tutorial for learning the Python programming language by making tests pass. Most tests are fixed by filling the missing parts of assert functions. Eg: self.assertEqual(__, 1+2). There is about 300 koans to do and I have completed 12 (3%) and 1 (out of 37) lessons. It get's harder as you complete them but nonetheless, it seems like a very good practise.
